  • First Week Exercise
  • Master Mobile UX Notes
  • Mobile First design will be simple and beautiful
  • So many more devices (2-3) per household
  • Downsides: expensive to do a full rebuild
  • Upsides: Best experience across devices, addresses mobile size constraints, future friendly
  • Desktop computer sales declining
  • People using phones a lot
  • Start small and scale up
  • Mobile is not a fad
  • "A lot of times, people don't know what they want until you show it to them"
  • Be there -- Be Useful -- Be Quick
  • Mobile Device use:
  • Reachability matters - make it as simple as possible to users
  • Speed Matters - reduce image size, webfonts, minify CSS and JS,GZIP files
  • Networks Matter
  • Placement matters - addresses reachability and creates impressing of improved speed
  • Perception matters - look for what will improve the experience
  • Make it simple, so people can use it easily
  • Mobile forms - put the labels inside the forms, float labels go to the top, reduce # of fields
  • Enable autocomplete in forms
  • Calls Matter - don't copy and paste a phone number
